# movie-website-info-recordbook-v3
本仓库用于记录互联网上已公开的影视类网站基础信息，重点面向电影与电视剧相关站点进行资料整理与留存。通过统一结构记录站点名称、访问地址、站点定位说明及基础分类信息，方便集中管理与长期查阅。本项目不涉及任何影视资源的存储、播放、下载或解析，仅对公开可获取的信息进行整理记录。所有内容均来源于公开网络环境，整理过程保持客观中立，不对站点内容质量、合法性或服务稳定性作出任何评价。仓库内容仅用于资料整理与研究参考，不构成任何形式的推荐或引导。
